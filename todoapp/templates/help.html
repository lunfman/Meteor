Deadlines

In how many days task should be completed
Task name + In (number)
number - how many days till deadline
Nb! Max value of In number is 90!
Examples:
Task1 In 5
Result
Task1 should be done in 5 days from today

Task1 By (january-december)
month - by which month

Examples:
Task1 By april

Result
Task1 should be done by 1 april

Task1 By december

Task1 should be done by 1 december

Task name By (1-30)
number - date of the current month

Examples:
Task1 By 5

Result
Task1 should be done by currents month day 5
For example if now is september and I use Task1 By 5 =>(5 september)

Task name By keyword
Keywords: 'tomorrow', 'next week', 'next month'
Examples
task By tomorrow
task By next week
task By next month

Task name By day(1-30) month(january-december) number - date of the month month - by which month Examples:
